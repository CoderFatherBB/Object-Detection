## About

This repository contains object-detection work for real-time industrial part counting. It includes input/testing videos, training notebooks, prediction code, and generated model run outputs.

## Repository Map

- `Training/`: training notebook, prediction script, frame extraction script, and YOLO run outputs.
- `Sample input video/`: sample input footage.
- `Testing Video/`: testing footage.


## Getting Started

Clone the repository:

```bash
git clone https://github.com/CoderFatherBB/Object-Detection.git
cd Object-Detection
```

Open the folder or notebook you want to explore. Most directories are self-contained, so start from the local `README.md`, source file, notebook, or project folder for that topic.
